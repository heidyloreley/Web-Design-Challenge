Summary: Latitude vs X

The purpose of this project was to analyze, “What’s the weather like as we approach the equator?”. This means, how cities’ weather changes as they are closer to the equator. To accomplish this analysis, it required to get a random sample of data using the OpenWheaterMap API to assemble a dataset of over 500 cities. This process considered cities along the globe, using latitudes from the lowest point in the southern hemisphere up to highest one in the northern hemisphere (from -90° to 90°) and with a longitude form -180° to 180° (from west to east).

After assembling the dataset, the data was ordered using python and pandas languages and finally plotting the relationship between latitude and 3 other variables (humidity, cloudiness and wind speed) with Matplotlib.

This site provides visualization and explanation of the information gather of the 547 selected cities, analyzing the relationship between the latitude and humidity, cloudiness and wind speed.
